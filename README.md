# Familien App

Eine einfache, live-syncende Familien-Website mit Kalender, To-Do-Liste und Mehrspieler-Spielen.

## Start lokal

1. `npm install`
2. `npm start`
3. Öffne `http://localhost:3000`

## Deployment

Die App ist für einfache Hosting-Plattformen wie Render, Railway oder Vercel (mit Server-Adapter) vorbereitet.
Die Server-Logik läuft mit Express und Socket.io direkt über `server.js`.
